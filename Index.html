HTML

html, a Linguagem da Web: Uma Base Essencial

HTML (HyperText Markup Language) é a linguagem fundamental para a construção de páginas web. Imagine-a como o esqueleto de um website, definindo a estrutura, organização e conteúdo de cada página. Através de tags, como <p> para parágrafos, <h1> para títulos e <img> para imagens, o HTML define a aparência e o layout da página.

Para entender melhor, pense em um documento de texto simples. O HTML adiciona "marcas" (tags) que indicam ao navegador como interpretar e exibir o conteúdo. É como dar instruções para o navegador: "Este é um título", "Este é um parágrafo", "Esta é uma imagem".

Por que HTML não é uma linguagem de programação?

Embora o HTML seja essencial para a web, ele não é considerado uma linguagem de programação. Isso porque o HTML não possui a capacidade de realizar cálculos, controlar o fluxo de execução ou tomar decisões. Ele apenas define a estrutura e o conteúdo da página.

A vida de um desenvolvedor HTML:

Desenvolvedores HTML são responsáveis por criar a estrutura e o layout das páginas web. Eles trabalham com o HTML para organizar o conteúdo, definir a aparência dos elementos e garantir a acessibilidade e usabilidade do site.

Um exemplo prático:
Imagine que você está criando um site de receitas. O HTML define a estrutura da página, como:

  • Um título principal: "Receitas Deliciosas"
  • Uma seção para a lista de receitas: "Receitas de Sobremesas"
  • Uma seção para cada receita individual, com título, ingredientes e instruções.
O HTML não define como a receita é preparada ou como o site interage com o usuário. Essa funcionalidade é adicionada por outras linguagens, como JavaScript e CSS.

em resumo:

O HTML é a base da web, definindo a estrutura e o conteúdo das páginas. Ele não é uma linguagem de programação, pois não possui a capacidade de executar ações complexas. Desenvolvedores HTML são essenciais para criar a estrutura e o layout dos websites, garantindo uma experiência online agradável e organizada.

python

Python é uma linguagem de programação muito popular e versátil, usada em diversas áreas como desenvolvimento web, ciência de dados e machine learning.

Quem estuda Python pode trabalhar em áreas como:

  • Desenvolvedor Python: Cria aplicações web, softwares e ferramentas usando Python.
  • Cientista de Dados: Analisa dados, cria modelos preditivos e extrai insights usando Python.
  • Engenheiro de Machine Learning: Desenvolve algoritmos de aprendizado de máquina para tarefas como reconhecimento de imagens e tradução automática.

Aprender Python abre portas para um mercado de trabalho em constante crescimento, com salários competitivos e oportunidades desafiadoras.

JavaScript

JavaScript é uma linguagem de programação usada para adicionar funcionalidades às páginas da web, como manipular janelas do navegado. Em alguns casos, JavaScript pode ser usado de forma intrusiva para realizar ações que você pode achar irritantes. Para esses casos, você pode desativá-lo completamente. Se você fizer isso, algumas páginas não funcionarão como planejado.

Para alterar as configurações de JavaScript:

  • Vá em Configurações.
  • Clique em Avançado na barra lateral esquerda e clique em Privacidade e segurança.
  • Em Privacidade e segurança, clique em Configurações do site.
  • Clique em JavaScript.
  • No topo, ative ou desative Permitido (recomendado).
  • Clique no botão Adicionar para definir preferências específicas do site.

Pessoas que estudam e trabalham com JavaScript são chamadas de desenvolvedores web. Eles usam JavaScript para criar interações dinâmicas e funcionalidades complexas em páginas da web, como animações, jogos e aplicativos web.
